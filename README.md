# Data and Machine Learning for Artistic Practice

This repo hosts teaching materials for the module Data and Machine Learning for Artistic Practice (DMLAP), and it is addressed to students of the MA/MFA Computational Arts course of Goldsmiths, University of London.


[The Original repo can be found here](https://github.com/IriniKlz/DMLAP/tree/main)

## Usage

```
git pull iriniklz main
```

Then in VIM

### Opetion 01
- Just accept the default message and close so the merge completes:
- Press Esc (to be sure you’re in normal mode).
- Type :wq and press Enter.
- :wq = write (save) + quit.
- The merge will complete with the default message (e.g. “Merge branch 'main' of …”).

### Option 2 – Edit the message first
- If you want to change the message:
- Press i to enter insert mode.
- Edit the top line (the commit message).
- Press Esc, then type :wq and press Enter.
